# Parking system (backend)

### Copy .env.example to .env

(update the values in .env)

```bash
cp .env.example .env
```

### Install dependencies

```bash
yarn install
```

### Run the server in development mode

```bash
yarn dev
```

### Run the server in production mode

```bash
yarn build
yarn start
```

### Run the tests

```bash
yarn test
yarn test:integration
```
